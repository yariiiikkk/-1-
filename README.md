"Проект на 1с"

Студент 4 курса Козлов Ярослав, направления "Программная инженерия"

О проекте:
Учебный проект, созданный в рамках обучения в вузе. Разработка прикладного решения для автоматизации кадрового учета организации.

Что реализована:
1.Справочники: Сотрудники, Отделы
2.Документы: ПриемНаРаботу, Увольнение, Отпуска
3.Отчет: КолРаботников
4.Подсистемы: ОтделКадров, Организация
5.Роли: Админ, ОтделКадров, Сотрудник
6.Журнал документов: КадровыйУчет
7. Регистры Накопления: Работники

Технологии:
- 1С:Предприятие 8.3
- Встроенный язык 1C
- СКД
- Запросы

Скриншоты:
1. Главный экран(sreenshots/glavniy)
2. Отчет(screenshots/otchet)
